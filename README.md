## Descripción del Proyecto

Este proyecto aborda el estudio y desarrollo de estructuras de datos basadas en árboles, un componente esencial en la construcción de sistemas de software eficientes. A través del análisis de su comportamiento algorítmico y sus aplicaciones en la industria, se busca desarrollar un criterio técnico sólido que permita seleccionar la estructura más adecuada según el contexto y los requerimientos del sistema.

El trabajo se organiza en dos etapas: una fase de investigación donde se documentan las características, complejidades y casos de uso de cada estructura mediante fichas técnicas, y una fase práctica donde se lleva a cabo la implementación funcional de una de ellas. El objetivo es que el estudiante no solo comprenda la teoría detrás de la notación Big O, sino que sea capaz de aplicarla para comparar el rendimiento real de distintas topologías arbóreas, identificando tanto sus fortalezas como sus limitaciones en situaciones concretas.

## Instrucciones de Compilación y Ejecución

Java BST: Esta instrucción carga el bytecode de la clase principal en la Máquina Virtual de Java (JVM). Al ejecutarse, la JVM reserva el espacio inicial de memoria para la raíz del árbol y dispara el flujo automatizado que demuestra, de manera secuencial y en tiempo real, las operaciones de inserción, búsqueda, recorridos y la reestructuración de nodos tras una eliminación.
